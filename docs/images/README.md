# スクリーンショット / デモ画像の置き場

README で参照する画像をここに置きます。以下のファイル名で保存してください（README がこの名前で参照）:

| ファイル名 | 内容 | 推奨 |
|-|-|-|
| `hero.png` | 完成パネルのフルスクリーン表示（一番の“映え”カット） | 1920×480 等、実際のサブモニタ表示。**最重要** |
| `editor.png` | アプリ内エディタでパネル編集中の画面 | パレット・配置・プロパティが見える全体像 |
| `demo.gif` | ライブ更新のデモ（数値が動く／アニメGIF素材が動く） | 5〜10秒・軽量（数MB以内）。ループ |
| `assets.png` | 素材ライブラリ（パック自動分類タブ）※任意 | カスタマイズ性の訴求 |

## キャプチャのコツ
- **hero**: 実際に使っている完成パネルを present（フルスクリーン）表示して撮る。これが告知の顔になる。
- **demo.gif**: [ScreenToGif](https://www.screentogif.com/) などで画面録画→GIF。数値が更新される様子・アニメGIFゲージが動く様子を短く。容量は圧縮して数MBに抑える。
- 個人情報（PC名・IP・アカウント名等）が写り込まないよう注意。
